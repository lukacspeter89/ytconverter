Your video is compressed in your browser — never uploaded, never stored on a server. Large files are read lazily (by byte range), so the whole file is never loaded into memory. On first run the ffmpeg.wasm core loads (~31 MB, cached afterwards). The finished file downloads with a _small suffix.
